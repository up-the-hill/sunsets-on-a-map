# Sunsets

A full-stack web app for mapping sunsets.

## Tech Stack

### Frontend
-   **Framework:** React 19 + Vite
-   **Styling:** Linaria
-   **Maps:** MapLibre GL JS
-   **Search:** Stadia Maps Search Box

### Backend
-   **Server:** Hono (Node.js)
-   **ORM:** Drizzle ORM
-   **Database:** PostgreSQL + PostGIS
-   **Classification Model:** TensorFlow.js (Node)
-   **Storage:** AWS S3
-   **Language:** TypeScript

### Infrastructure
-   **Containerization:** Docker
-   **SSL:** Certbot (Let's Encrypt)
-   **Reverse Proxy:** Nginx
